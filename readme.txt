About:

A desktop application to keep track of your daily tasks. Allows up to 15 to-dos with a title and text area. To-dos can be added, marked complete, and removed.
Custom coloring is supported for each to-do section, as well as text color inversion.


Functions:

Complete button (green check mark): Marks your to-do as complete. This locks the section and adds a check mark background.
The button can be toggled to undo the complete status.

Delete button (red X): Permanently removes a to-do.

Settings (blue gear button): Opens a window with options to invert text color and select the color for the current to-do section.
